阮一峰GIT命令

上班

  • 1. 进入项目切换到github的主分支，比如 master。 "git checkout master"
  • 2. 更新/同步代码。 "git pull origin master"
  • 3. 回到自己分支。 "git checkout '自己分支名'"
  • 4. 合并代码。 "git merge master"

下班

  • 1. 查看所有未 add 的代码。 "git start"
  • 2. 添加到本地仓库。 "git add ."
  • 3. 添加到远程仓库。 "git commit -m '注释'"
  • 4. 重复上班4步（防止有人中途提交代码导致冲突）。
  • 5. 提交到远程仓库。 "git push origin master"
